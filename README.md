# Проект “MESTO”

[Ссылка на проект ](https://pr9nek.github.io/mesto-project/)

Cервис для обмена фотографиями, похожий на социальную сеть. 
Пользователь может добавлять фото-карточки мест, в которых был или хочет побывать.

Технологический стек:
* HTML
* CSS
* JavaScript

Реализованные задачи:
1. Создал структуру проекта.
2. Настроил сборку с помощью Webpack.
3. Настроил транспиляцию с помощью Babel.
4. Реализовал адаптивный интерфейс под устройства с шириной экрана 1280px и 320px.
